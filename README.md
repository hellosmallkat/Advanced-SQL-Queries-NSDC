## Overview

This repository contains my solutions and work for the **Advanced SQL Queries** project developed by the **National Student Data Corps (NSDC)**.

Using the **New York City Airbnb** dataset, this project explores increasingly complex SQL techniques to answer real-world business questions through data analysis.

The goal is to strengthen analytical SQL skills while practicing industry-standard querying techniques that are commonly used by data analysts, business analysts, and data scientists.

---

## Learning Objectives

Through this project, I practiced:

* Writing complex SQL queries
* Combining data from multiple tables
* Using aggregate functions
* Filtering and grouping data
* Working with subqueries
* Creating Common Table Expressions (CTEs)
* Applying window functions
* Ranking and partitioning data
* Producing analytical reports
* Improving query readability and organization

---

## Dataset

The project uses a **New York City Airbnb** dataset containing information about:

* Listings
* Hosts
* Neighborhoods
* Prices
* Reviews
* Availability
* Property characteristics

The dataset provides realistic business scenarios for practicing SQL-based data exploration and reporting.
